VBA excel projects

Application to track daily departments progress and tasks for one of factoring company. This application is prepared to work as cloud application in random folder. It contains three levels of data gathering and presentation. On first level there are worksheets for each working group which allows to enter daily work progress. On second level there are worksheets to daily synchronization of all entered data for each group. Synchronisation is done with power query with dynamic path and variables. Worksheet writes all data to separate sheet as archive of all daily progress. This function is programed with VBA. On this level information are also gathered from two other sheets. All daily information are presented as report about current group progress, presence and KPI score. On third level all information from group are gathered in separate report to management board. On this level progress in the implementation of the yaerly, monthly and daily plan is presented in form of tables and graphs.

Application to send tax information to clients . This application allows to register, edit or delete accountant office clients with details about them needed in daily office work. Second function is to prepare files with client information for each office worker. In this sheets tax information are gathered for each month. Third option is to send mails to email address from clients database with tax information entered by accountants. Application allows to send mail for group of client with summary of tax information. Mail are prepared in application and exported to MS Outlook. There each mail can be modify if needed and send. Mail content can be also modify in application. Application keeps track of email dates, send status and tax information for each client and accountants. This application works in Microsoft Windows 365 Business Cloud. Application was made with usage of Power Query and VBA.
